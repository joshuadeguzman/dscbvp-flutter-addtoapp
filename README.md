# Flutter Examples

> A demonstrable application to showcase Flutter's Add-to-App feature for Android


## Setup

Clone the repo

```
$ git clone https://github.com/joshuadeguzman/dscbvp_add_to_app
```

Install dependencies

```
$ flutter packages get
```

Compile and install the app on a connected device

```
$ cd feedly_android
$ ./gradlew installDebug
```

Run the standalone Flutter app

```
$ cd flutter_module
$ flutter run -v
```

## Development

Run the Android project from Android Studio by clicking the Play button

<br/>

Attach the Flutter app

```
$ flutter attach
```
